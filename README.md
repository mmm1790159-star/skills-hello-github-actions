# Notification System

## Description
A modular notification system featuring CRM integration and support for SMS and WhatsApp messages.

## Features
- **CRM API Integration**: Manage API requests to your CRM seamlessly.
- **SMS (Pars Green)**: Send SMS notifications using the Pars Green service.
- **WhatsApp (Twilio)**: Enable WhatsApp messaging via Twilio.
- **Unified Manager**: Centralized logic to coordinate notifications.

## Installation
1. Clone this repository using:
   ```bash
   git clone https://github.com/mmm1790159-star/skills-hello-github-actions.git
   ```
2. Switch to the `notification-system` branch:
   ```bash
   git checkout notification-system
   ```

## Usage
- Ensure Python (>=3.7) is installed.
- Run the `main.py` script:
  ```bash
  python main.py
  ```
